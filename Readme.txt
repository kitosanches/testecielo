-O projeto de front end foi desenvolvido com Angular +2 e Karma/Jasmine para os testes

-É necessário iniciar ambos os projetos cursomc e teste-cielo.

-O projeto de front-end pode ser acessado em http://localhost:4200

-Os testes podem ser executados utilizando o comando ng test e clicando em DEBUG na tela que irá se abrir

-No projeto é possível consultar categorias, produtos e clientes. È possível cadastrar apenas clientes.
Obs: Clientes cadastrados no banco H2 irão sumir após a aplicação ser parada.
